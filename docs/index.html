KSchema Table Action Generator

Quick Start

npm install -g @keshavsoft/kschema-api-gen-actions npx @keshavsoft/kschema-api-gen-actions ShowAll

Generated Structure

project/ ├── end-points.js │ ├── ShowAll/ │ ├── controller.js │ ├── dal.js │ ├── route.js │ └── validation.js

Architecture Flow

CLI ↓ parseInput.js ↓ resolveCommand.js ↓ loadCommand.js ↓ actions.json ↓ dynamic import() ↓ tasks/actions/ ↓ template generation

Example actions.json

[ { "cmd": "showAll", "file": "showAll", "exportFile": "ShowAll", "group": "GetMethods", "description": "Generate ShowAll GET action" } ]

Supported Commands

Command Method Description
ShowAll GET Generate ShowAll table action
Insert POST Generate Insert table action
Distinct GET Generate Distinct table action

Adding New Actions

1. Create action file tasks/actions/update.js 2. Add config entry { "cmd": "update", "file": "update", "exportFile": "Update", "group": "PostMethods", "description": "Generate Update POST action" } 3. Done
